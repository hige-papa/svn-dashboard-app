# アプリケーション ヘルプページ

このページでは、本アプリケーションの主な機能と使い方について説明します。

## 1. はじめに

このアプリケーションは、カレンダー、備品、施設、ユーザーなどの情報を効率的に管理するためのツールです。日々の業務やイベントの管理をスムーズに行うことができます。

## 2. ログインとログアウト

### ログイン

アプリケーションを利用するには、まずログインが必要です。
1.  サインインページ (`/signin`) にアクセスします。
2.  登録済みのユーザー名とパスワードを入力し、「ログイン」ボタンをクリックします。

### ログアウト

アプリケーションの利用を終了する際は、画面上部またはサイドバーにある「ログアウト」ボタンをクリックしてください。

## 3. 主要機能の利用方法

### 3.1. カレンダー機能

カレンダー機能では、イベントの登録、表示、管理を行うことができます。

#### イベントの表示

カレンダーは、日次、週次、月次の3つのビューで表示できます。画面上部のナビゲーションを使って、表示を切り替えることができます。

-   **日次ビュー**: 特定の1日のイベントを時間軸で確認できます。
-   **週次ビュー**: 1週間分のイベントを一覧で確認できます。
-   **月次ビュー**: 1ヶ月分のイベントを俯瞰して確認できます。

#### イベントの登録

新しいイベントを登録するには、カレンダー画面の「イベント登録」ボタンをクリックするか、カレンダー上の空いている日時をクリックします。

イベント登録フォームでは、以下の情報を入力できます。

-   **タイトル**: イベントの名称。
-   **日時**: イベントの開始日時と終了日時。
    -   **単一イベント**: 特定の1日のみのイベント。
    -   **期間イベント**: 複数日にわたるイベント。開始日と終了日を設定します。
    -   **繰り返しイベント**: 定期的に発生するイベント。
        -   **繰り返しパターン**:
            -   **毎日**: 毎日繰り返します。
            -   **毎週**: 毎週繰り返します。特定の曜日を選択することもできます（例: 毎週月曜日と水曜日）。
            -   **毎月**: 毎月繰り返します。特定の日付（例: 毎月15日）または特定の週の曜日（例: 毎月第2火曜日）を選択できます。
            -   **毎年**: 毎年繰り返します。
            -   **平日**: 月曜日から金曜日まで繰り返します。
        -   **終了条件**:
            -   **特定の日付まで**: 繰り返しを終了する日付を指定します。
            -   **特定の回数まで**: 繰り返しを終了する回数を指定します。
            -   **無期限**: 繰り返しを無期限に設定します。
-   **場所**: イベントが開催される場所。
-   **説明**: イベントの詳細情報。
-   **優先度**: イベントの重要度。
-   **参加者**: イベントに参加するユーザー。
-   **関連施設**: イベントで使用する施設。
-   **関連備品**: イベントで使用する備品。

#### イベントの編集・削除

登録済みのイベントをクリックすると、詳細が表示されます。詳細画面からイベントの編集や削除を行うことができます。

### 3.2. 備品管理

備品管理機能では、組織内の備品を登録・管理できます。

-   **備品一覧**: 登録されている備品の一覧を表示します。
-   **備品登録**: 新しい備品を登録します。
-   **備品詳細・編集**: 各備品の詳細情報を確認し、必要に応じて編集します。

### 3.3. 施設管理

施設管理機能では、組織内の施設（会議室、研修室など）を登録・管理できます。

-   **施設一覧**: 登録されている施設の一覧を表示します。
-   **施設登録**: 新しい施設を登録します。
-   **施設詳細・編集**: 各施設の詳細情報を確認し、必要に応じて編集します。

### 3.4. ユーザー管理

ユーザー管理機能では、アプリケーションを利用するユーザーを管理できます。

-   **ユーザー一覧**: 登録されているユーザーの一覧を表示します。
-   **ユーザー登録**: 新しいユーザーを登録します。
-   **ユーザー詳細・編集**: 各ユーザーの詳細情報を確認し、必要に応じて編集します。

### 3.5. その他の機能

-   **出席管理**: ユーザーの出席状況を管理します。
-   **経費管理**: 経費の登録と管理を行います。
-   **メンテナンス**: システムのメンテナンスに関する情報を確認します。
-   **売上管理**: 売上データを管理します。
-   **設定**: アプリケーションの各種設定を行います。
-   **タスク管理**: タスクの登録と進捗管理を行います。
-   **チーム管理**: チームの情報を管理します。

## 4. よくある質問 (FAQ)

-   **Q: イベントがカレンダーに表示されません。**
    -   A: イベントの期間が現在表示しているカレンダーの範囲内にあるか確認してください。また、繰り返しイベントの場合は、繰り返しルールが正しく設定されているか確認してください。
-   **Q: ログインできません。**
    -   A: ユーザー名とパスワードが正しいか確認してください。大文字・小文字の区別にも注意してください。問題が解決しない場合は、システム管理者にお問い合わせください。
